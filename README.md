# Random Password Generator

**Автор:** Ачабаев Билял

## Краткое описание

Random Password Generator — это графическое приложение для генерации случайных паролей с настраиваемыми параметрами. Позволяет выбрать длину пароля, типы символов (строчные/заглавные буквы, цифры, спецсимволы), копировать пароль в буфер обмена и сохранять историю сгенерированных паролей.

### Требования:
- Python 3.6 или выше
- Стандартная библиотека Python (tkinter, json, os)

### Запуск программы:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd RandPassGen
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```
